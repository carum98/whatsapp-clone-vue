**Whatsapp clone** build with Vue 3 + Typescript + Vite

## SetUp
This project use [Chat API](https://github.com/carum98/chat-api)

With Docker
```bash
docker run -d -p 8080:8080 ghcr.io/carum98/whatsapp-clone:[tag]
```
